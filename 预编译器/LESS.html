Less是一门CSS预处理语言，它扩展了CSS语言，增加了变量，Mixin、函数等特征，使CSS更容易维护和扩展。
Less可以运动在Node或浏览器端。

- 定义一个变量，变量名相同且是同级变量的时候，后面定义的变量会覆盖前面定义的变量
- 同级同便签下定义属性用& 取地址符
- 引入外部文件 import    “../common/var”    
注意：1.import后面不用加冒号。2.不加格式后缀
.clear:after {
    content：‘’；
    display：block；
    clear：both；
    height：0；            主要这四个属性必须的    
}

- 写Less函数格式的时候必须加 . / # XXXX，例如 .fn-test 和 #fn-test，参数必须是@XXX。
- 写Less变量的时候必须@width： 200px；
- &：extend(.header)；继承来自   .header   的所有属性

.wrapper {

    width: @_width;

- 类似群主选择器的继承函数

.header {
    width: 1000px;
    margin: 0 auto;
}
.banner {
    &:extend(.header);
    height: 500px;
}  


@import "../common/fn";
@import "../common/var";

    }

.box {
    .fn-flex(30%);

- fn.less

.fn-text-over(){
    content: '';
    display: block;
    clear: both;
}
.fn-bor-radius(@size){
    border-radius: @size;
}
.fn-flex(@xixi){
    width: @xixi;
    display: flex;
    flex-wrap: wrap;
}
